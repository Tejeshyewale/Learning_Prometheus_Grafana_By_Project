# Flask Monitoring Stack using Kubernetes, Prometheus & Grafana

## Overview

This project demonstrates deploying a Flask application on Kubernetes using Minikube and monitoring it with Prometheus and Grafana.

## Technologies Used

* Flask
* Docker
* Kubernetes (Minikube)
* Prometheus
* Grafana
* Helm

## Steps Performed

1. Created and containerized a Flask app using Docker.
2. Deployed the application on Minikube using Kubernetes manifests.
3. Installed Prometheus using Helm and configured it to scrape Flask metrics.
4. Installed Grafana and connected it with Prometheus for dashboard visualization.

## Features

* Containerized Flask application
* Kubernetes deployment
* Metrics monitoring with Prometheus
* Interactive dashboards using Grafana

## Outcome

Successfully implemented a complete monitoring and observability pipeline for a Flask application in a Kubernetes environment.
